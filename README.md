Here, we will explain to you how to start the program.
First make sure you have all files necessary to play. You should have : "area.c", "command.c", "id.c", "manage_shape.c", "project.c", "shape.c", "area.h", "id.h", "project.h", "Makefile".

What the program is doing ?

    The program allows you to draw figures in an area to create a drawing. You can create points, lines, squares, rectangles, circles and polygons.can.

How to start the program ?

    First you have to know the path where our program is in your computer.
    If you don't know how to get the path :

    1) If you're on Mac :

        a. Go in the Finder and got to the folder which contains our program.

        b. Right click on the folder and click on "new terminal at folder"

    2) If you're on Windows :

        a. Go to your Documents

        b. In your Documents go to where our program is and type 'cmd' in the bar at the top of the window.

        c. You are good to go.

    Now let's start it :

        1. Enter in the terminal : "make". This command compile the program and create and executable.

	  2. Enter in the terminal : "draw.exe"

        3. Start to draw and enjoy ! (You will be asked the size of the area)
